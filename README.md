# Integrated-Student-Management-Portal
EduHub is a dynamic, multi-user school management system designed to digitize academic administration. The portal handles the entire student lifecycle from secure login and profile management to course registration and result viewing providing a centralized hub for educational data.
Core Functionalities & Technical Implementation
Role-Based Access Control (RBAC): Engineered a secure backend logic in PHP to differentiate user roles, ensuring students and staff only access authorized modules.

Dynamic Course Registration: Built a relational database system in MySQL to manage student-course enrollments, preventing duplicate entries and ensuring data consistency.

Automated Result Processing: Developed server-side scripts to calculate and display academic performance metrics (GPA/Grades) based on database records.

Profile Management: Implemented a secure interface for users to update personal information, including image upload handling and data validation.
